Ans 1 In Python, data types are used to classify and store different kinds of data. 

An Integer (int) is used to store whole numbers without any decimal point, such as 10, 25, or -5.

A Float (float) is used to store numbers that contain decimal values, such as 3.14, 5.5, or -2.75.

A String (str) is used to store text or a sequence of characters enclosed within single or double quotation marks, such as "Hello" or "Python".

A Boolean (bool) data type stores only two values: True or False, and is commonly used in decision-making and conditional statements
